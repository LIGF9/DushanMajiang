# 独山麻将

一个基于Pygame开发的独山麻将游戏，支持人机对战和AI自动对战。

## 游戏特色

- 🎮 **完整的麻将游戏体验**：支持碰、杠、胡等基本操作
- 🤖 **智能AI对手**：多种AI策略，可选择不同难度
- 🎵 **丰富的音效和背景音乐**：沉浸式游戏体验
- 🎨 **精美的游戏界面**：多种主题和角色选择
- 📊 **游戏历史记录**：保存每局游戏结果，支持查看历史对战
- ⚙️ **灵活的游戏设置**：可调整游戏参数、AI思考时间等

## 安装说明

### 环境要求

- Python 3.8+  
- Pygame 2.0+

### 安装依赖

1. 克隆或下载项目到本地
2. 打开命令行终端，进入项目根目录
3. 运行以下命令安装所需依赖：

```bash
pip install -r requirements.txt
```

## 运行游戏

### 方式一：直接运行可执行文件

在 `dist` 目录下找到 `majiang.exe`，双击即可运行。

### 方式二：从源码运行

```bash
python majiang.py
```

## 游戏规则

### 基本规则

- **牌型**：使用标准麻将牌，包括万、条、筒各1-9，共108张牌
- **人数**：4人游戏
- **庄家**：第一局随机选择庄家，之后由上一局赢家坐庄
- **手牌**：每人初始13张牌，通过摸牌、打牌、碰杠等操作进行游戏
- **胡牌**：组成符合规则的牌型即可胡牌

### 独山麻将特色规则

- **鸡牌**：贵州地区麻将特色捉鸡玩法，增加游戏趣味性
- **番数计算**：根据胡牌牌型和胡牌方式计算番数，可在settings中调整

## 游戏操作

### 主菜单

- **开始游戏**：进入游戏界面
- **设置**：调整游戏参数
- **历史对局**：查看过往游戏记录

### 游戏中

- **鼠标左键**：
  - 点击手牌：选择要打出的牌
  - 点击操作按钮：执行碰、杠、胡等操作
  - 点击设置按钮：打开游戏设置

- **键盘操作**：
  - `ESC`：返回主菜单
  - `Space`：快速出牌（使用AI推荐）

## 开发说明

### 核心模块

1. **GameManager**：负责游戏逻辑的核心管理，包括游戏流程控制、玩家管理等
2. **UIManager**：负责游戏界面的渲染和用户交互
3. **Player**：玩家类，包含玩家的手牌、操作等信息
4. **Tile**：麻将牌类，定义了牌的属性和行为
5. **SoundManager**：音效管理，负责播放背景音乐和游戏音效
6. **majiangAI**：AI算法实现，提供不同难度的AI对手

### 扩展开发

1. **添加新AI策略**：在 `majiangAI.py` 中添加新的AI类
2. **修改游戏规则**：编辑 `rule.py` 文件
3. **添加新角色**：在 `resource/avatar/` 目录下添加新头像
4. **修改界面样式**：编辑 `ui_manager.py` 和相关资源文件

### 打包游戏

使用PyInstaller打包游戏：

```bash
pyinstaller majiang.spec
```

## 游戏设置详解

### 基本设置

- **人类思考时间**：玩家考虑出牌的最长时间（秒）
- **AI思考时间**：AI考虑出牌的最长时间（秒）
- **初始分数**：游戏开始时每个玩家的初始分数
- **测试轮数**：测试模式下自动进行的游戏局数

### 显示设置

- **显示所有牌面**：是否显示其他玩家的手牌
- **显示玩家名称**：是否在游戏中显示玩家名称
- **显示AI版本号**：是否显示AI玩家的版本信息

### 音效设置

- **播放背景音乐**：是否开启背景音乐
- **背景音乐音量**：调整背景音乐音量
- **播放游戏音效**：是否开启游戏音效
- **游戏音效音量**：调整游戏音效音量
- **播放打牌读牌音效**：是否播放出牌时的语音提示
- **打牌读牌音效音量**：调整语音提示音量

### 高级设置

- **测试模式**：AI自动对战，无需人工操作
- **游戏加速模式**：加快游戏进程，适合测试
- **默认再来一局计时**：游戏结束后自动开始下一局的等待时间

## 历史记录

游戏会自动保存每局游戏的结果，包括：

- 玩家信息和分数变化
- 每轮出牌记录
- 胡牌牌型和番数
- 游戏时长

可以在主菜单的"历史对局"中查看详细记录。

## 常见问题

### Q: 游戏运行卡顿怎么办？
A: 可以尝试关闭一些不必要的特效，或降低游戏分辨率。

### Q: 如何调整游戏音量？
A: 在游戏设置中可以调整背景音乐和音效音量。

### Q: 游戏支持联机对战吗？
A: 目前只支持单机对战，后续版本可能会加入联机功能。

### Q: 如何修改游戏规则？
A: 可以编辑 `rule.py` 文件中的规则实现。
